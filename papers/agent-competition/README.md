# Agent Competition Paper Pack

This folder stores a small paper pack for LitNavigator's agent-focused competition track.

Selection date: 2026-06-16

## Included papers

1. ReAct: Synergizing Reasoning and Acting in Language Models
   Source: https://arxiv.org/abs/2210.03629
   Why: core reasoning-plus-acting loop for agent control flow.

2. Toolformer: Language Models Can Teach Themselves to Use Tools
   Source: https://arxiv.org/abs/2302.04761
   Why: tool-use decisions and API invocation behavior.

3. Reflexion: Language Agents with Verbal Reinforcement Learning
   Source: https://arxiv.org/abs/2303.11366
   Why: reflection, retry, and language-level feedback loops.

4. Generative Agents: Interactive Simulacra of Human Behavior
   Source: https://arxiv.org/abs/2304.03442
   Why: memory, reflection, and planning primitives.

5. Voyager: An Open-Ended Embodied Agent with Large Language Models
   Source: https://arxiv.org/abs/2305.16291
   Why: long-horizon behavior and reusable skill acquisition.

6. A Survey on Large Language Model Based Autonomous Agents
   Source: https://arxiv.org/abs/2308.11432
   Why: agent taxonomy and concept-map backbone.
