# OW-3.1 — DISCOVER relevance gate + multilingual query Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Steps use `- [ ]`.

**Goal:** Stop `find-sources` from digesting off-topic sources, and make it work for non-English goals — the two P0 gaps the 10-scenario e2e found (source relevance ~44%; non-English 0/4).

**Architecture:** Two cheap-LLM seams added to the DISCOVER orchestrator, each with a deterministic offline pass-through: (1) **query normalization** — rewrite any-language goal into a concise English search query used for the English-biased adapters + cosine rerank; (2) a **relevance gate** — after ranking, a cheap LLM drops sources that are not actually about the goal (the only thing that knows "Megalopolis (film)" ≠ Raft consensus), keeping ≥`min_keep` so we never starve digest. The user's original `goal_text` is preserved for intent + downstream teaching language.

**Tech Stack:** Python, `litnav.discover.*`, `litnav.llm.router` (cheap tier, metered), `pytest`. Baseline at plan time: **279 passed** (post PR-#6 merge). No new model tier.

```python
import sqlite3
from litnav.storage.schema import init_db
from litnav.discover.contract import Source
from litnav.discover import relevance
from litnav.llm import router

def _src(t, a=""): return Source(source_type="web", source_id=t, title=t, url="u", abstract=a, authority_score=0.5)

def test_offline_passthrough(monkeypatch):
    monkeypatch.setenv("LITNAV_LLM_PROVIDER", "none")
    c = sqlite3.connect(":memory:"); init_db(c)
    srcs = [_src("A"), _src("B")]
    assert relevance.relevance_gate("topic", srcs, conn=c, session_id="s") == srcs  # unchanged offline

def test_drops_irrelevant_keeps_order(monkeypatch):
    monkeypatch.setattr(router, "complete_json", lambda *a, **k: {"relevant_indices": [0, 2]})
    c = sqlite3.connect(":memory:"); init_db(c)
    srcs = [_src("Raft paper"), _src("Megalopolis film"), _src("Paxos paper")]
    out = relevance.relevance_gate("raft consensus", srcs, conn=c, session_id="s", min_keep=1)
    assert [s.title for s in out] == ["Raft paper", "Paxos paper"]

def test_never_starves(monkeypatch):
    monkeypatch.setattr(router, "complete_json", lambda *a, **k: {"relevant_indices": []})
    c = sqlite3.connect(":memory:"); init_db(c)
    srcs = [_src("A"), _src("B"), _src("C")]
    out = relevance.relevance_gate("topic", srcs, conn=c, session_id="s", min_keep=2)
    assert len(out) == 2 and out[0].title == "A"   # all-dropped -> keep top min_keep by rank
```
