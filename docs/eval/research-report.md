## 2. Knowledge gaps

- BKT/Rasch overstatement (load-bearing): the primary keypoint mastery path is a hand-tuned kp_bump + linear kp_confidence=0.3*correct_obs, with no fitted slip/guess/transit and no Rasch theta (irt_theta is a schema column never read/written); the spec §11/§12 honestly says 'BKT-lite/estimate' but the reviewer-facing per-step tables (methods ④d, storyboard ⑥) assert plain 'BKT — Corbett & Anderson 1995'. This is the single biggest claim-vs-implementation gap.
- Two literal docstring-vs-code gaps a reviewer will catch by reading one function: make_distractors claims 'overgenerate-and-rank' but takes the first-n (no rank step), and flaw_gate is called 'SAQUET-style' but screens ~3 structural flaws vs SAQUET's ~19 item-writing flaws.
- RefD mis-attribution and GraphRAG mislabel: refd.py is corpus co-occurrence/reference-distance asymmetry, not Liang 2015's Wikipedia link-graph RefD; concept extraction is tagged 'GraphRAG-style' though GraphRAG is query-focused summarization. ACE (JEDM 2024) is the correct precedent for the actual two-signal pattern.
- No global-consistency / cycle-breaking pass over prereq edges (GKROM/DGCPL show this is the accuracy frontier); LitNavigator scores edges only locally/pairwise, and single-source digest produces 0 surviving prereq edges.
- Learning-science levers are named but dormant: spaced retrieval probes are enqueued (review_queue/retention_log) but never delivered before teaching; worked examples are a static label with no fading ladder; interleaving is unused despite concept_edges.similarity providing a principled trigger. Testing effect is used for grading, not for strengthening.
- Orchestration depth: ReAct+Plan-and-Solve is topology-only (implicit branching, no logged plan artifact) and reteach has no Reflexion-style error memory (counter-driven strategy flip). Escalation objective (pedagogical-error cost near the mastery band) is correct but the trigger is hand-set, not calibrated from logged data.
- Grading throws away signal: grade_kp emits score_0_5 + confidence but drives mastery off the BINARY correct flag; no component-extraction step and no self-consistency (k-sample) near the threshold, despite the 0-5 scale being where human-LLM alignment peaks.
- Open-web live edge accuracy is unvalidated (spec risk A); LitNavigator cannot claim benchmark-grade accuracy and should report a live self-measured edge-accuracy number instead of closed-corpus numbers.
- Citation/venue errors for ICCSE submission: RouteLLM is ICLR 2025 (methods says 'ACL 2025'); ReAct is ICLR 2023 / 2022 preprint; the 'specialised KT always beats LLM' framing overstates arXiv:2603.02830 (which shows LLM-KT narrowly top).
